### 9.1 Fibonacci Style / Climbing Stairs

**চিনবেন কীভাবে:** dp[i] শুধু আগের ১–২টা মানের ওপর নির্ভর করে — "কত উপায়ে i-তে পৌঁছানো যায়"

**Demo: Climbing Stairs** — [LC 70](https://leetcode.com/problems/climbing-stairs/) _(Easy — Google/Amazon High)_

**Statement (Demo):** `n`টি ধাপ বিশিষ্ট একটি সিঁড়ি দেওয়া — প্রতিবার ১টি বা ২টি ধাপ উপরে উঠতে পারলে, চূড়ায় পৌঁছানোর মোট কতটি অনন্য উপায় আছে বলুন।
উদাহরণ: `n=3` → `3` (1+1+1, 1+2, 2+1) | ⚡ `1 ≤ n ≤ 45`

**Approach:** i-তম ধাপে আসা যায় i-1 (এক লাফ) বা i-2 (দুই লাফ) থেকে — তাই `dp[i] = dp[i-1] + dp[i-2]`। শুধু শেষ দুটো মান লাগে বলে O(1) space।

```js
function climbStairs(n) {
  let prev = 1,
    cur = 1; // dp[0], dp[1]
  for (let i = 2; i <= n; i++) [prev, cur] = [cur, prev + cur];
  return cur;
}
```

**Complexity:** Time O(n), Space O(1)
