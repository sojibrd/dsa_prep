### 9.6 Edit Distance

**চিনবেন কীভাবে:** "transform one string to another", insert/delete/replace-এর min সংখ্যা — 2D DP, three operations min

**Demo: Edit Distance** — [LC 72](https://leetcode.com/problems/edit-distance/) _(Hard — Google/Microsoft High)_

**Statement (Demo):** দুটো string `word1` ও `word2` দেওয়া — `word1` থেকে `word2`-এ রূপান্তর করতে ন্যূনতম কয়টি অপারেশন লাগবে বলুন (অনুমোদিত অপারেশন: insert, delete, replace)।
উদাহরণ: `word1="horse", word2="ros"` → `3` (h->r replacement, r deletion, e deletion) | ⚡ `len(word1), len(word2) ≤ 500`

**Approach:** `dp[i][j]` = word1-এর প্রথম i → word2-এর প্রথম j বানানোর খরচ। অক্ষর মিললে খরচ নেই (diagonal); নাহলে replace/delete/insert-এর min + 1। Base: খালি string → j inserts বা i deletes।

```js
function minDistance(w1, w2) {
  const m = w1.length,
    n = w2.length;
  const dp = Array.from({ length: m + 1 }, (_, i) =>
    Array.from({ length: n + 1 }, (_, j) => (i === 0 ? j : j === 0 ? i : 0)),
  );
  for (let i = 1; i <= m; i++)
    for (let j = 1; j <= n; j++)
      dp[i][j] =
        w1[i - 1] === w2[j - 1]
          ? dp[i - 1][j - 1]
          : 1 +
            Math.min(
              dp[i - 1][j - 1], // replace
              dp[i - 1][j], // delete
              dp[i][j - 1],
            ); // insert
  return dp[m][n];
}
```

**Complexity:** Time O(m·n), Space O(m·n)
