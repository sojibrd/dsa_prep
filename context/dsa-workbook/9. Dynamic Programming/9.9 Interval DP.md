### 9.9 Interval DP

**চিনবেন কীভাবে:** "burst balloons", "matrix chain", subarray/interval-এর ভেতরে **শেষ operation কোনটা** ভেবে ভাঙা — dp[l][r], ছোট interval থেকে বড়

**Demo: Burst Balloons** — [LC 312](https://leetcode.com/problems/burst-balloons/) _(Hard — Google/Meta High)_

**Statement (Demo):** `n`টি বেলুন দেওয়া আছে যেখানে `nums[i]` = i-তম বেলুনের স্কোর। বেলুনগুলো ফাটালে `nums[i-1] * nums[i] * nums[i+1]` পাওয়া যায়। সব বেলুন ফাটিয়ে সর্বোচ্চ কত স্কোর করা সম্ভব বলুন।
উদাহরণ: `nums=[3,1,5,8]` → `167` | ⚡ `n ≤ 300`

**Approach:** উল্টো করে ভাবুন — (l, r) খোলা interval-এ **সবার শেষে** কোন বেলুন k ফাটবে? তখন তার প্রতিবেশী হবে l আর r নিজেই। `dp[l][r] = max(dp[l][k] + a[l]·a[k]·a[r] + dp[k][r])`। দুই পাশে 1 padding দিন।

```js
function maxCoins(nums) {
  const a = [1, ...nums, 1];
  const n = a.length;
  const dp = Array.from({ length: n }, () => new Array(n).fill(0));
  for (let len = 2; len < n; len++) {
    for (let l = 0; l + len < n; l++) {
      const r = l + len;
      for (let k = l + 1; k < r; k++) {
        // k = শেষে ফাটানো বেলুন
        dp[l][r] = Math.max(dp[l][r], dp[l][k] + a[l] * a[k] * a[r] + dp[k][r]);
      }
    }
  }
  return dp[0][n - 1];
}
```

**Complexity:** Time O(n³), Space O(n²)
